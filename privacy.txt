Privacy Policy

The following data are collected by Genshin Assistant

• How you interact with the app. For example which features you do and don't use. This is sent to my analytics provider.
• Steps you took before a crash or error occur. This causes a bug report to be sent to my analytics provider. 

This data is collected using Google Analytics and Microsoft App Center.

The data sent does not include any personal information. Only things like Device type, Android version and Country etc.
You are completely anonymous and none of the data collected can personally identify you.
